Detailed Description

Library header file for the radio libraries to control radio chips.

Author
Matthias Hertel, http://www.mathertel.de
Copyright
Copyright (c) 2014-2015 by Matthias Hertel.
This work is licensed under a BSD style license.
See http://www.mathertel.de/License.aspx

This library enables the use of diverse radio chips by sharing the same class definition. Implementation for the following Radio Chips are available:

  • RDA5807M
  • SI4703
  • TEA5767

The following chip is planned to be supported too: ...

More documentation and source code is available at http://www.mathertel.de/Arduino

History:

  • 08.07.2014 creation of the common radio class.
  • 15.07.2014 examples working with RDA5807M.
  • 26.08.2014 examples working with SI4703.
  • 31.08.2014 Doxygen style comments added.
  • 05.02.2015 mainpage content added.
